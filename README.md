## 🍽️ Exploratory Data Analysis of Zomato Data
This project performs Exploratory Data Analysis (EDA) on a Zomato dataset to uncover insights about restaurants, cuisines, ratings, and customer preferences. Using Python libraries, the analysis explores patterns in restaurant distribution, pricing, and ratings, providing a foundation for business decisions and predictive modeling.

## 📊 Project Overview
- Clean and preprocess Zomato dataset.
- Perform EDA to identify trends and patterns.
- Visualize restaurant distribution across cities and cuisines.
- Analyze ratings, votes, and cost for two.
- Generate insights into customer preferences and restaurant performance.

## 🛠️ Tools & Technologies
- Python 3
- Pandas – data manipulation
- NumPy – numerical operations
- Matplotlib & Seaborn – data visualization
- Jupyter Notebook – interactive analysis

## 📂 Repository Contents
- EDA_Zomato.ipynb – Jupyter Notebook with complete EDA workflow.
- zomato.csv – dataset used for analysis.
- README.md – project documentation.

## 🔍 Key Insights
- Popular cuisines and their distribution across cities.
- Relationship between ratings, votes, and restaurant cost.-
- Identification of top-performing restaurants.
- Visualization of restaurant density in major cities.
